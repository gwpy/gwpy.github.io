    Citing GWpy¶

    If you have used GWpy as part of a project that leads to a scientific publication, please acknowledge this by citing the GWpy paper: D. M. Macleod et al, SoftwareX, 13, 100657 (2021). The citation in various formats is given below:

    @article{gwpy,
    title = "{GWpy: A Python package for gravitational-wave astrophysics}",
   author = {{Macleod}, D.~M. and {Areeda}, J.~S. and {Coughlin}, S.~B. and {Massinger}, T.~J. and {Urban}, A.~L.},
  journal = {SoftwareX},
   volume = 13,
    pages = 100657,
     year = 2021,
     issn = {2352-7110},
      doi = {10.1016/j.softx.2021.100657},
      url = {https://www.sciencedirect.com/science/article/pii/S2352711021000029},
}
