    Installation¶

    Installing GWpy¶

    Conda¶

    The recommended way of installing GWpy is with Conda:

    conda install -c conda-forge gwpy

    Pip¶

    python -m pip install gwpy

    All of these will be installed using any of the above install methods.

    GWpy also depends on the following other packages for optional features:

    • glue.ligolw: to read/write EventTable with LIGO_LW XML format (see LIGO_LW XML)

    • LDAStools.frameCPP: to read/write data in GWF format

    • nds2: to provide remote data access for TimeSeries (see gwpy-timeseries-remote)

    • uproot: to read/write EventTable with ROOT format (see ROOT)
